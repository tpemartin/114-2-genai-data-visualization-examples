# Iterative prompting for dataset documentation

  1. 請分析檔案，列出所有欄位名稱 (Column Names)、資料類型 (Data Types)（例如字串、整數、浮點數），並根據資料內容為每個欄位提供一個初步的定義描述及範例值
  2. 請說明每個數值變數的單位（如：公尺、美金）、取值範圍，以及是否有特定的分類代碼（例如：0 代表女性，1 代表男性）。同時，請識別出哪些欄位可能存在缺失值 (Missing Values) 或異常邏輯。
  3. 為這個數據集生成一份正式的 Dataset Documentation (README)。內容須包含：
      數據集概述：資料來源與收集目的。
      資料來源：數據集的來源機構，平台或出版機構。
      欄位字典：包含名稱、定義、單位、格式、範例值。
      已知限制：資料的偏誤或不適用的分析情境。
      建議研究問題：基於現有變數可進行的 3 個分析方向。
