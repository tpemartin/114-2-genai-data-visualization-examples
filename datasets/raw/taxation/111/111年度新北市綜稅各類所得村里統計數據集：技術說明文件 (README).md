### 新北市 111 年度各村里綜合所得稅各類所得統計資料集 (README)

#### 1\. 數據集概述 (Dataset Overview)

在現代都市治理與社會科學研究中，村里級所得數據具備不可替代的戰略重要性。透過精確至基層行政區劃的財稅統計，政府得以落實「量能課稅原則」，並以此作為城鄉發展規劃、公共基礎設施布點及社會福利資源分配的關鍵決策依據。尤其民國 111 年度（2022 年）正值全球供應鏈調整與後疫情時代經濟復甦之關鍵期，本資料集為觀測新北市各地方產業韌性（如五股區製造業聚落與板橋區服務行政中心之消長）提供了微觀的經社特徵證據。本數據集核心資訊彙整如下：

* **資料來源：**  財政部財政資訊中心。  
* **收集目的：**  提供新北市各鄉鎮市區及村里之綜合所得稅核定資料，旨在建構新北市基層社會經濟發展之觀測基準。  
* **統計範圍：**  涵蓋 111 年度各納稅單位申報核定之各類所得細目，包括營利、薪資、利息、股利及稿費等。數據透明化之核心目標在於優化資源配置之邊際效益，精準對準社會脆弱性高之區域，從而提升治理效能。為確保後續數據分析之嚴謹性，以下針對欄位定義進行詳細說明。

#### 2\. 欄位字典 (Data Dictionary)

精確的欄位解讀是防止統計誤判、確保跨區域對比一致性的基石。在處理村里級所得數據時，研究者必須嚴格區分稅務名詞之定義，以避免重複計算或低估實際經濟活動。

| 欄位名稱 | 定義與格式 | 單位 |  
| ----- | ----- | ----- |  
| **縣市別** | 行政區劃之首級單位，固定為新北市（String） | N/A |   
| **村里** | 基層行政區劃單位（String） | N/A |  
| **納稅單位(戶)** | 該行政區內申報綜合所得稅之總戶數（BigInt） | 戶 |
| **合計** | 該村里申報之各類所得總額總計（BigInt） | TWD (千元) |
| **營利所得** | 獲自營利事業之盈餘所得（BigInt） | TWD (千元) |
| **薪資所得** | **薪資淨所得** ：勞務報酬扣除稅法扣除額後之課稅基礎（BigInt） | TWD (千元) |
| **薪資收入** | **薪資總額** ：指納稅義務人實際取得之勞務毛收入（BigInt） | TWD (千元) |
| **股利所得** | 獲自公司、合作社之股利或分配盈餘（BigInt） | TWD (千元) |
| **利息所得** | 各類存款（非郵政儲金）、公債及債券之利息所得（BigInt） | TWD (千元) |
| **執行業務所得** | 律師、會計師等專門職業之勞務淨收益（BigInt） | TWD (千元) |
| **租賃及權利金** | 出租財產或專利權、商標權之收益（BigInt） | TWD (千元) |
| **財產交易所得** | 出售財產（如非分離課稅之不動產）之利得（BigInt） | TWD (千元) |
| **機會中獎所得** | 競技、競賽或機會中獎之獎金或獎品（BigInt） | TWD (千元) |
| **退職所得** | 一次或分期領取之退休金、資遣費（BigInt） | TWD (千元) |
| **其他所得** | 不屬於上述分類之各類所得（BigInt） | TWD (千元) |
| **稿費所得** | 稿費、版稅及演講費之合計（BigInt） | TWD (千元) |
| **稿費收入** | 稿費類項目的毛收入總額（BigInt） | TWD (千元) |
| **申報大於歸戶** | 原始申報數與稅務歸戶後之差額，常用於觀察跨區申報或複雜家戶結構（BigInt） | TWD (千元) |  
**技術性評註：**  分析人員應特別留意「所得」與「收入」之技術差異：所得（Taxable Income）係稅務核定之課稅基礎，而收入（Gross Revenue）則更接近家戶的現金流入。此外，「股利所得」與「執行業務所得」之比例，常被視為衡量特定村里經濟活力的關鍵指標。

#### 3\. 已知限制 (Known Limitations)

識別統計數據中的「盲點」對於維持分析的客觀性至關重要。本資料集係基於稅務申報資料，並非完整之家戶可支配所得調查，故在使用時應注意以下邊界：

* **非納入所得項目：**  根據財政部規範，本統計未包含以下項目：  
* **政府移轉支出：**  如中低收入戶補助、育兒津貼及各類社會保險給付。  
* **免稅所得：**  如證券交易所得（停徵期間）、土地交易所得等。  
* **分離課稅所得：**  包含郵政儲金利息、短期票券利息及特定金融產品之扣繳利得。  
* **資料偏誤評估：**  
* **高所得層估計偏低：**  因資本利得（如證券交易）多採免稅或分離課稅，導致高資產集中村里的實際經濟實力往往高於稅務合計數。  
* **社會脆弱區域失真：**  資料未計入政府補貼，在分析如深坑或石碇等傳統區域或高齡化村里時，可能無法完整反映居民的實際生存資源與基本購買力。  
* **適用建議：**  本數據集應定位為「課稅能量觀察」，而非「單一家戶購買力」之精確度量衡，不宜直接推論為該區居民的所有經濟來源。

#### 4\. 建議研究問題 (Suggested Research Questions)

透過將靜態財稅指標轉化為動態的經社命題，可大幅提升本資料集的應用深度。基於 111 年度之數據分布，建議研究方向如下：

1. **所得結構分布與階層特徵分析：**  對比薪資所得與股利所得之比例。例如，觀察哪些區域屬於「純勞動導向」（薪資比例顯著），哪些區域則因資本利得（股利、利息）集中而呈現資產富集特徵，藉此建立村里層級的所得分配不均指標。  
2. **區域經濟差距與新興開發區衝擊：**  以林口區「南勢里」（合計逾 174 億 TWD）對比三重區之傳統村里。研究高成長區域（如林口、新莊頭前、昌平里）如何與周邊地區形成所得梯度，探討都市開發誘發的「經濟極化現象」。  
3. **城鄉所得彈性與地理環境相關性：**  研究偏遠行政區（如石碇、烏來、坪林）與核心開發區的所得彈性差異。透過分析利息與股利所得較高村里的地理分布，探討高端住宅區對基層經濟數據的拉動作用，以及邊際村里是否面臨資源分配的邊際效益遞減。本研究框架旨在鼓勵使用者在尊重隱私與數據邊界的原則下，深入解析數據背後的社會實相，為新北市的永續治理提供具備研究信度的洞察。
